A deep learning–powered web application that identifies common potato leaf diseases from user-uploaded images using a Convolutional Neural Network (CNN). Built with Python, TensorFlow/Keras, Flask, HTML/CSS/JavaScript, and optionally Streamlit for rapid prototyping. Achieved ~97% accuracy in classifying Early Blight, Late Blight, and healthy leaves.
